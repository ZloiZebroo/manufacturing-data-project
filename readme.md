# Manufacturing data collection and analysis project 
## Technologies:
- Postgres
- Grafana
- Python
- Telegram API
- OPC UA

## Quick start: <P>
Set properties in .env file <P>
Build project <P>
`./build.sh` <P>
Run project <P>
`./run.sh` <P>
Stop project <P>
`./run.sh` <P>
View logs <P>
`docker compose logs -f` <P>
